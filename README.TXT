工程器材:
	stm32f103c8t6最小系统板
	
工程目的:
	实现对5mmLED灯光强度控制，不同灯光模式切换，能够和手机上位机相连。
	
硬件资源:
	1,5mmLED灯
	2,HC-05蓝牙模块和手机蓝牙相连
	  从而实现PWM输出控制LED亮度.
	3,光敏电阻检测光照强度.
	4,1602显示屏显示系统当前模式和光照强度，
实验现象:
	通过按键或者安卓上位机改变光照模式，光照模式包括手动和自动模式，自动模式还包括走廊模式、教室模式、厕所模式.				                   		
	
注意事项:
	1.光照强度范围在0-899.
	2.手机app滑动条控制灯光强度时存在一定的bug.
